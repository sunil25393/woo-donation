== Description ==

woo donation:

Woo Donation Plugin is used to add Donation in woo-commerce cart and extend woo-commerce functionality.It easy to configure and use.  By using this plugin You can allow user to add doantion in cart.

Option Available in Plugin: 
 - Create Donation Product
 - Add Donation to cart automatically 
 - Make donation mandatory
 - Set minimum Amount to donation
 - Shortcode to show donation form on custom page
 - Allowed to change all lable in form
 - Simple and easy to configure

For Detailed Documentation <a href="http://sunilprajapati.in/woo-donation/">Click Here</a>. 
 
Dont forget to RATE.

Note: This plugin is only compatible with woocommmece.

== Installation ==

Steps to Install Plugin.

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Woocommerce -> Donation ->  screen to configure the plugin.
4. GO to checkout page  -> Donation Form will be displayed.
